Implementation of Modified HTC Using VUMAT Subroutine

This repository contains an Abaqus UMAT/VUMAT implementation of a modified Hsieh–Ting–Chen (HTC) equivalent strain formulation integrated within an enhanced local damage framework. The model is designed to simulate quasi-brittle fracture behavior under uniaxial tensile loading conditions.
